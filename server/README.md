# FarmEasy Backend

FastAPI and MCP backend for the FarmEasy corn leaf disease model bundle.

## What this gives you

- `POST /predict` for Flutter/mobile apps using multipart image upload.
- `POST /predict/base64` for JSON-only clients.
- `POST /predict/video` for uploaded video files; the backend samples frames.
- `WS /ws/live` for live Flutter camera frames over WebSocket.
- `GET /metadata` for classes, selected model weights, thresholds, and MCP info.
- `GET /docs` and `GET /redoc` generated by FastAPI.
- `/mcp` mounted as a Streamable HTTP MCP server for MCP-compatible AI clients.
- `python -m server.mcp_server` for standalone stdio or HTTP MCP usage.

## Setup

From the project root:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r server/requirements.txt
```

Run the API:

```bash
python -m server --host 0.0.0.0 --port 8000
```

Open:

- FastAPI Swagger docs: http://localhost:8000/docs
- ReDoc docs: http://localhost:8000/redoc
- MCP Streamable HTTP endpoint: http://localhost:8000/mcp

## Quick Test

```bash
curl http://localhost:8000/health
curl http://localhost:8000/metadata
curl -F "file=@/path/to/leaf.jpg" http://localhost:8000/predict
curl -F "file=@/path/to/scan.mp4" "http://localhost:8000/predict/video?sample_every_n_frames=15"
```

All image, video-frame, and live-frame predictions run the leaf gate first.
The disease classifier is called only when `is_leaf=true`.

## Standalone MCP

For local MCP clients that prefer stdio:

```bash
python -m server.mcp_server --transport stdio
```

For standalone Streamable HTTP MCP:

```bash
python -m server.mcp_server --transport streamable-http
```

The standalone HTTP MCP endpoint is `http://127.0.0.1:8001/mcp` by default.
Set `FARMEASY_MCP_HOST` and `FARMEASY_MCP_PORT` to change it.

## Configuration

Copy `server/.env.example` into your deployment environment and set values as needed.
The backend auto-detects `FarmEasy_Integration_Bundle/` first and then `model/`.
The default model is `effb3` because this bundle includes `effb3_best.pth`.
Only set `FARMEASY_MODELS=all` if all referenced checkpoint files are present.

Full API details are in `server/docs/API.md`.
